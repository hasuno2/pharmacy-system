# pharmacy-system
## Overview 
A simple system to manage drug information, client interactions, and prescription tracking using an Excel database.

## Features (Planned) 
- Drug database (Excel)
- Search/filter drugs
- Client registration
- GUI interface (Tkinter)
- Export functionality

## Tech Stack 
- Python 3.x
- pandas
- openpyxl
- Tkinter (GUI)
- GitHub Projects (coordination)
